Schema Control Pro

Version 1.5.0

Schema Control Pro is a WordPress plugin for defining, prioritizing, and injecting custom JSON-LD structured data rules. Target rules by post type, taxonomy, singular page, or homepage, and control how your schemas coexist with AIOSEO output.

== Features ==

- Rule-based schema injection per page type, taxonomy, or individual post
- Priority and mode system to control override vs. merge behaviour
- Full JSON-LD graph management with ID normalization and reference rewriting
- AIOSEO coexistence: suppress, merge, or override third-party schema output
- Import / export rules and schemas as JSON
- Admin UI for creating and managing rules without code
- Built-in validation and diagnostics for structured data output
